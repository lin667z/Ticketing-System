# 铁路互联网购票身份核验须知

   为了进一步完善铁路实名制购票工作，自2014年3月1日起，中国铁路客户服务中心网站[https://www.12306.cn](https://kyfw.12306.cn/otn/gonggao/12306.cn)（以下简称本网站）将对互联网注册用户和乘车人进行身份信息核验。

## 一、 注册用户和乘车人

**1. 身份信息核验**

   注册用户可在“个人信息”中“查看个人信息”中，查看注册用户的身份信息核验状态（以下简称状态）。

   乘车人可在“常用信息管理”中“乘车人”中，查看乘车人的身份信息核验状态（以下简称状态）。

   状态种类及含义：

   1.1 “已通过”：指注册用户、乘车人的身份信息已经通过国家身份认证权威部门核验或铁路部门人工核验，其中姓名、证件类型、证件号码、国籍等身份信息不可修改。

   1.2 “待核验”：指注册用户、乘车人的身份信息在核验过程中。

   1.3 “未通过”：指注册用户、乘车人的身份信息经过核验但未通过，请按照证件原件的身份信息修改。 

   待核验和未通过的用户无法购票，如需办理业务，需持有效身份证件原件到车站售票窗口办理核验。办理注册人身份核验时，请持在12306.cn网站注册时填写的有效身份证件原件。办理乘车人身份信息核验时，请持乘车人有效身份证件原件及注册人注册时填写的有效身份证件原件，到车站售票窗口办理核验。

   外国护照用户和乘车人还可通过12306网上传证件照片方式核验。

   遇有姓名超长、生僻字、繁体字等情形的，仅可在车站售票窗口办理，还要提供注册手机号后4位或注册邮箱前3位。

**2. 添加乘车人**

   “已通过” 注册用户均可在本网站正常添加乘车人，其中中华人民共和国居民身份证（简称居民身份证）、中华人民共和国港澳居民居住证（简称港澳居民居住证）、中华人民共和国台湾居民居住证（简称台湾居民居住证）、外国人永久居留身份证、港澳居民来往内地通行证、台湾居民来往大陆通行证、中华人民共和国护照（简称中国护照）、外国人护照（简称外国护照）注册用户可添加居民身份证、港澳居民居住证、台湾居民居住证、外国人永久居留身份证、港澳居民来往内地通行证、台湾居民来往大陆通行证、中国护照、外国护照的乘车人。

   “待核验”、“未通过”注册用户不能添加乘车人。

   “已通过”的乘车人在核验通过后30天内不能删除。

**3. 网上购票、退票、改签**

   3.1 网上购票

   “已通过”注册用户可为“已通过”乘车人购票。“未通过”和“待核验”注册用户和乘车人不可购票。

   3.2 网上退票

   注册用户均可为已购车票正常办理网上退票。

   3.3 网上改签

   注册用户可为“已通过”的乘车人办理网上改签。

**4. 车站换取报销凭证、退票、改签**

   4.1 “已通过”乘车人的所购车票，均可在车站售票窗口换取报销凭证、改签、退票。

   4.2 “已通过”乘车人姓名、证件号码与本网站填写完全一致的，如为可自动识读证件，可在自动售（取）票机换取报销凭证。

   4.3 “已通过”乘车人，遇有姓名超长、生僻字、繁体字等情形的，所购车票应到车站售票窗口凭证件原件和订单号码办理换取报销凭证、改签、退票。

## 二、其他相关事项

**1．身份证件有效性**

   有效身份证件是指在有效期内的中华人民共和国居民身份证、港澳居民居住证、台湾居民居住证、外国人永久居留身份证、按规定可使用的有效中国护照和外国护照、港澳居民来往内地通行证、台湾居民来往大陆通行证。

**2．乘车人数量**

   注册用户名下的乘车人累计不超过30人（上限人数会根据春运、暑运等具体情况进行调整）。

**3. 委托办理核验**

   “待核验”、“未通过”的注册用户或乘车人可委托他人到车站售票窗口办理身份信息核验。办理时，需同时携带代办人和委托人的有效身份证件原件。

**4．网上用户姓名输入**

   在本网站上输入用户姓名时，请按其有效身份证件原件上的姓名准确填写，遇有下列情况时，按以下规则填写:

   （1）居民身份证、港澳居民居住证、台湾居民居住证、中国护照按姓名顺序填写中文姓名。

   （2）外国人永久居留身份证按姓名顺序填写中文或英文姓名。

   （3）港澳居民来往内地通行证、台湾居民来往大陆通行证按姓名顺序输入简体中文姓名。

   （4）外国护照按姓名顺序填写英文姓名。姓、名中有空格的应输入空格，姓和名之间也应使用空格分隔。

   （5）确认姓名中生僻字无法输入时，可用生僻字拼音或同音字替代。

   （6）输入姓名保存后，遇有系统无法正确显示的汉字，可用该汉字的拼音或同音字重新修改后保存。

   （7）姓名中有繁体字无法输入时，可用简体替代。

   （8）姓名中有“.”或“•”时，请仔细辨析身份证件原件上的“.”或“•”，准确输入。

   （9）姓名较长，汉字字符合计超过30个（1个汉字算2个字符）的，需按姓名中第一个汉字开始按顺序连续输入30个字符（空格字符不输入）。外国人永久居留身份证和外国护照使用英文姓名的，应按姓名顺序全部填写。

**5．居民身份证无法自动识读**

   持居民身份证、港澳居民居住证、台湾居民居住证、外国人永久居留身份证用户在车站售票窗口办理身份信息核验时，居民身份证不能通过自动识读设备自动读取的，不予办理，需到发证机关换证后再予办理。

**6.注册用户被抢注**

   本网站提示“身份信息重复”的注册用户，须注册用户本人（不可代办）持在网站注册时填写的有效身份证件原件到车站售票窗口办理或注册人通过“手机APP”在线按提示办理。
